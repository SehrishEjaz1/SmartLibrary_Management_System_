## 📖 Overview
A full-stack web application to manage digital books featuring both user and admin functionalities. Built with Node.js, React, and SQL Server.

---

## 🛠️ Tech Stack
- **Backend:** Node.js, Express.js
- **Frontend:** React.js, HTML5, CSS3
- **Database:** Microsoft SQL Server
- **Version Control:** Git, GitHub

---

## ✨ Features

### For Users
- **Search Books:** Check book availability instantly
- **Bookmark Books:** Save favourite books for quick access
- **Download Books:** Download available ebooks
- **View Fines:** Track fines for overdue books

### For Admins
- **Book Upload:** Upload new ebooks to the system
- **Manage Books:** Edit or delete uploaded books
- **Manage Users:** View user activity and feedback
- **Top 3 Books:** Display most popular books by engagement

---

## 🖥️ Installation
```bash
# Backend
cd backend
node index.js

# Frontend
cd frontend
npm install
npm start
```
App runs at: `http://localhost:3000`

Login as User or Admin and explore the digital library experience.
